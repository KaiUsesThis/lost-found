# Lost and Found

Helping you find lost items.
[![Build Status](https://travis-ci.org/KaiUsesThis/lost-found.svg?branch=master)](https://travis-ci.org/KaiUsesThis/lost-found)

## To run:
```
export FLASK_APP=lostfound:app
flask run -h 0.0.0.0 -p 8080
```

## To run tests:
```
$ python runTest.py
```
